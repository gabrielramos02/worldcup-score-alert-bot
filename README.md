# 🌍 World Cup Score Alert Bot
#### This README was created with Copilot AI. I'm not expressive enough.
A Telegram bot that provides real-time FIFA World Cup 2026 information and score alerts directly in your chat.

## ✨ Features

- **Team List** - View all participating teams in the FIFA World Cup 2026
- **Score Alerts** - Get notifications for live match updates *(coming soon)*
- **Team Information** - Fetch detailed information about specific teams *(in development)*
- **Easy Setup** - Simple configuration with environment variables

## 🛠️ Tech Stack

- **Language**: Python 3.13+
- **Bot Framework**: [telegram-py](https://github.com/telegram-py/telegram-py)
- **API**: [ESPN Sports Core API](https://github.com/pseudo-r/Public-ESPN-API)
- **Package Manager**: [uv](https://github.com/astral-sh/uv)

## 📋 Prerequisites

Before you begin, ensure you have:

- Python 3.13 or higher
- A Telegram Bot Token (get one from [@BotFather](https://t.me/botfather))
- [uv](https://github.com/astral-sh/uv) (recommended) or pip
